# QR Code Generator Pro

A professional, beautiful web-based QR Code Generator built with simple HTML, Tailwind CSS, and plain JavaScript.

## Features
- **Multiple Types**: Generate QR codes for URLs and plain Text.
- **Customization**: 
  - Choose foreground and background colors.
  - Adjust the size (resolution) of the output.
  - Select Error Correction Levels (L, M, Q, H).
- **Live Preview**: See your settings applied immediately (upon clicking generate).
- **Download**: Export your QR code as a high-quality PNG.

## Technologies
- **HTML5**
- **Tailwind CSS** (via CDN) for styling.
- **QRCode.js** (via CDN) for generation logic.
- **Font Awesome** (via CDN) for icons.

## How to use
1. Clone the repository or download the files.
2. Open `index.html` in your browser.
3. Enter your URL or text, customize the look, and click **Generate**.
4. Download your QR code!
